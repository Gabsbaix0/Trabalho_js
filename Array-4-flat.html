JavaScript Array Methods

Array.flat()

O Array.flat() cria um novo array contendo todos os elementos de sub-arrays concatenados, isso significa que, se houver listas dentro de outras listas, elas serão "desempacotadas" e seus elementos serão inseridos em uma única lista, de maneira organizada, respeitando a profundidade definida.
O flat() também permite especificar a profundidade de níveis de aninhamento você quer aplanar.
