JavaScript Array Methods

Array.slice()

O slice() pega um pedacinho da sua lista de coisas e faz uma lista nova só com esse pedacinho. A sua lista original fica do mesmo jeito. É como tirar uma foto de uma parte do seu desenho, o desenho inteiro continua lá.
Você só precisa dizer onde começa o pedacinho que você quer copiar e onde ele termina (lembrando que o finalzinho não entra na cópia).
slice(start): Extrai do índice start até o final do array.
slice(start, end): Extrai do índice start até o índice end (excluindo o elemento no índice end).
